# LONA — Lokala Naturvårdssatsningen

## Vad är LONA?

LONA är Sveriges största satsning på att stimulera lokal naturvård. Programmet stödjer projekt som främjar naturvård, friluftsliv och folkhälsa. Kommunen är formell sökande — det är alltid kommunen som ansvarar juridiskt och ekonomiskt för att genomföra och redovisa projekten.

## Bidragsbelopp

- **Ordinarie projekt (stigar, ängar/betesmarker):** Upp till 50% finansiering av projektkostnader
- **Våtmarksprojekt:** Upp till 90% finansiering
- **Medfinansiering:** Ideellt arbete räknas som medfinansiering

## Process för Djurambulansen Syd

1. **Identifiera projekt:** Viltrehabilitering-projekt som passar LONA-kriterierna
2. **Kontakta kommunens naturvårdsansvarig:** Presentera projekt och sök stöd för kommunal ansökan
3. **Arbeta fram ansökan:** Genom Naturvårdsverket LONA-tjänst
4. **Kommunen ansöker:** Till Länsstyrelsen Blekinge eller Länsstyrelsen Skåne
5. **Länsstyrelsen beviljar eller avslår:** Beslut fattas enligt tilldelningsram

## Argument för viltrehabilitering inom LONA

- **Naturvård:** Minska lidande för skadat vilt genom professionell rehabilitering
- **Lokal kunskapsökning:** Djurambulansen Syd är redan etablerad expertresurs i regionen
- **Dokumenterad behov:** Försäljning av viltrehabilitering-tjänster visar på befintligt behov i Blekinge/Skåne
- **Folkhälsa & utbildning:** Rehabilitering kan kombineras med utbildningsaktiviteter för lokala skolors naturkunskapsundervisning
- **Friluftsliv:** Kan innefatta naturvårdsarbete på rehabiliteringsplatsen
